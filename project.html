Portfolio case

ArtPhone как демонстрационный e-commerce проект

Проект показывает, как учебный HTML/CSS/JavaScript магазин можно превратить в аккуратную коммерческую витрину Apple-техники без переписывания на React.

Смотреть главную Открыть каталог
Features

Что реализовано

  • Главная страница с hero, преимуществами, популярными товарами, категориями и промо-блоками.
  • Sticky glass-header, поиск с подсказками, мобильное меню и корзина.
  • Современные карточки товаров с бейджами, рейтингом, избранным, CTA и быстрым просмотром.
  • Гостевая корзина, toast-уведомления, scroll reveal и анимация добавления в корзину.
Stack

Технологии

  • HTML5 semantic layout.
  • CSS3, CSS variables, responsive grid, glass/blur effects.
  • Vanilla JavaScript, localStorage, DOM events, IntersectionObserver.
  • Static multi-page architecture без сборщика и фреймворка.
UX/UI

Дизайн-решения

  • Apple-like premium: светлый фон, воздух, крупные изображения, мягкие тени.
  • Marketplace-механики: hit/new/sale бейджи, старая цена, рейтинг, быстрые действия.
  • Mobile-first ограничения для сеток, кнопок, карточек и навигации.
  • Понятные CTA и коммерческие блоки для trade-in, рассрочки и доставки.
Value

Что демонстрирует case

  • Умение улучшать существующий проект без полного переписывания.
  • Работу с визуальной системой, адаптивом и микроинтеракциями.
  • Понимание коммерческого e-commerce UX и пути пользователя к покупке.
  • Аккуратную фронтенд-разработку на чистом HTML/CSS/JavaScript.
Business value

Почему этот проект полезен бизнесу

ArtPhone демонстрирует, как frontend-разработчик мыслит не только блоками и стилями, но и коммерческим сценарием: первое впечатление, доверие, выбор товара, сравнение, корзина и оформление заказа работают как единая воронка.
